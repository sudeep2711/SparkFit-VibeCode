# SparkFit — AI-Powered Personal Fitness Coach

SparkFit is a cross-platform mobile app that delivers a fully personalized fitness experience powered by generative AI. It replaces generic workout plans with a conversational onboarding flow, dynamically generated 7-day routines, and a context-aware AI coach available before, during, and after every workout.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React Native, Expo, TypeScript |
| Navigation | React Navigation |
| Backend | Supabase (PostgreSQL, Auth, Edge Functions) |
| AI / LLM | Google Gemini 2.5 Flash |
| Serverless Runtime | Deno (Supabase Edge Functions) |
| Platforms | iOS, Android, Web |

---

## Features

- **Conversational AI Onboarding** — Chat with the AI coach to define your goals, fitness level, available equipment, and schedule. The app extracts your profile through natural dialogue before generating your plan.
- **Personalized 7-Day Workout Plans** — Gemini generates a structured weekly routine tailored to your exact profile, with exercise names, sets, reps, and coach tips per movement.
- **Immersive Workout Execution** — Thumb-friendly stepper inputs (+/−), automated 90s rest timers, per-set timers, a global workout timer, and a top progress bar.
- **Live AI Coach Tips** — Dynamic coaching cues injected per exercise during the workout, pulled directly from the AI-generated plan.
- **Mid-Workout AI Chat** — Context-aware chat overlay accessible during an active workout. Automatically passes the current exercise and set to the AI for relevant, in-the-moment advice.
- **Workout Logging** — Full execution data (sets, reps, weight, duration) stored per session in Supabase with JSONB flexibility.
- **Progress Heatmap** — Visual calendar heatmap showing full (green) vs. partial (orange) completion based on logged workout data.
- **Standalone AI Coach Chat** — General fitness chat interface using your profile and active plan as system context for personalized answers.
- **Subjective Feedback Capture** — Post-workout perceived exertion rating and qualitative notes stored for future adaptive plan generation.

## Architecture

```
┌─────────────────────────────────────────────┐
│           React Native (Expo)               │
│  Auth → Onboarding → Dashboard → Workout    │
└────────────────────┬────────────────────────┘
                     │ @supabase/supabase-js
┌────────────────────▼────────────────────────┐
│              Supabase                       │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ Auth     │  │PostgreSQL│  │   Edge   │  │
│  │          │  │+ RLS     │  │Functions │  │
│  └──────────┘  └──────────┘  └────┬─────┘  │
└────────────────────────────────────┼────────┘
                                     │
                     ┌───────────────▼──────────────┐
                     │     Google Gemini 2.5 Flash   │
                     │  generate-workout-plan        │
                     │  chat-coach                   │
                     │  chat-onboarding              │
                     └──────────────────────────────┘
```

### Edge Functions
- **`generate-workout-plan`** — Takes onboarding transcript, queries user profile, calls Gemini, and upserts a structured 7-day plan into the database.
- **`chat-coach`** — General fitness chat with user profile + active plan injected as system context.
- **`chat-onboarding`** — Scoped conversational agent that extracts fitness parameters before triggering plan generation.

### Database Tables
- `profiles` — User metadata and AI context summary (1:1 with auth.users via Postgres trigger)
- `workout_plans` — JSONB 7-day plan generated by Gemini
- `workout_logs` — JSONB execution data per session
- `workout_feedback` — Perceived exertion and qualitative notes

Row Level Security (RLS) is enforced on all tables.

## Getting Started

### Prerequisites
- Node.js 20+
- Expo CLI (`npm install -g expo-cli`)
- Supabase account
- Google Gemini API key

### Setup

```bash
# Install dependencies
cd app && npm install

# Configure environment
# Add your Supabase URL, anon key, and Gemini API key to your Supabase Edge Function secrets

# Run the app
npx expo start
```
